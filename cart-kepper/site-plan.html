Overview

Purpose

This is a shopping list website this is something that you can make a list and pull it up and know what you need to buy. I want it to have a checklist and make it as done. I also want it to keep track of what they buy so that you can rebuy them. also have it add suggestions.

Audience

I know that this is something that moms and girls use. This is something that they made it was a piece of paper and then you forgot. This is so that you don't forget something it is the worst so you can just but it on your phone. You always have the phone with you so that you can put it on your phone so that they can use it and not forget something.

Dynamic elements

With this I want to have it have buttons so that you can add to the list. I want to make a to-do list type of thing to show that you need to buy it. I also have a search bar to find what you need to buy or create a new item.
